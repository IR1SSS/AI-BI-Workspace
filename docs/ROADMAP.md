# Roadmap

## Phase 1: File-Based MVP

- CSV, Excel, and JSON import.
- Parquet normalization.
- Data preview and profiling.
- Data quality issue detection.
- Cleaning plan preview and execution.
- Automatic dashboard generation.
- ECharts dashboard rendering.
- SQLBot-style ChatBI over the active dataset version.
- Dataset-specific LLM-generated suggested questions.

## Phase 2: Dashboard Authoring

- Manual chart selection and pinning.
- Save ChatBI answers as dashboard cards.
- Dashboard version history.
- Export dashboard snapshots.

## Phase 3: Database Sources

- PostgreSQL and MySQL connectors.
- Table and field discovery.
- Sampling and snapshot mode.
- Read-only direct query mode.
- SQL safety checker for external databases.

## Phase 4: Semantic Layer

- Metrics and dimensions.
- Business terms and field aliases.
- Human correction workflow.
- Multi-dataset retrieval.
- Optional vector search for business terminology.

## Phase 5: Operations

- Scheduled refresh.
- Background worker process.
- Report subscriptions.
- Audit log and LLM trace review.
- Workspace permissions.
